# youxiakewebapp
Vue仿游侠客webapp

#### 1.300ms点击延迟

`cnpm install fastclick --save`

#### 2.轮播

`cnpm install vue-awesome-swiper --save`

#### 3.动画

`cnpm install animate.css --save`

#### 4.倒计时秒杀插件

`cnpm install vue-countdown --save`

#### 5.axios

`cnpm install axios --save`
